# 2D Fighter (Local Desktop Game)

This project is a local (non-browser) 2D fighter game built with Python + `pygame`.

## Features
- Menu screen with a **Play** button
- Player vs CPU fight
- Side-to-side movement
- Two attacks: **punch** and **kick**
- Different character colors
- Health bars and fight end condition
- End screen with **Back to Menu** button

## Controls
- `A` / `D`: move left / right
- `J`: punch
- `K`: kick

## Run locally
1. Install Python 3.10+.
2. Install dependency:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the game:
   ```bash
   python main.py
   ```
